üöÄ Installation & Usage

Quick Start

bash ~/cerberus-kingdom-core/scripts/install.sh

This will:

  • Create directory structure
  • Install dependencies (Python, Flask)
  • Copy core components
  • Create startup scripts
  • Configure autostart

Start the System

~/start_kingdom.sh

This will start:

  • The Throne (auto-processing daemon)
  • Head 1 Sniffer (clipboard capture)
  • Head 4 Gatekeeper (truth filter)

Test It

echo "I fucking love this system" > ~/KINGDOM_ENGINE/inbox/test.txt

Check status:

~/kingdom_status.sh

üîå REST API

Status Endpoint

GET /status

Returns system status, processed count, uptime, and recent errors.

Accepted Files

GET /accepted

List all files classified as Truth or Fact.

Quarantined Files

GET /quarantine

List all files classified as Lies or blocked.

Recent Logs

GET /logs

Retrieve last 100 log entries.

Stop Daemon

POST /stop

Stop the auto-processing daemon.

Start Daemon

POST /start

Start the auto-processing daemon.
